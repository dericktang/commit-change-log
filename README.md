# commit-changelog
使用commit-changelog以后，git commit 的提交内容和格式会被限制，同时运行npm run changelog将其生成到changelog中


## Install
```shell
npm install commit-changelog -g
```

## Quick Start
``` shell
commitChangelog init
or
commitChangelog init npm/cnpm

git add 
git cz

npm run changelog
```
